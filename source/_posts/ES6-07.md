### Object
#### Object.keys()
```
	var json = {foo:"1",baz:"2"};
	Object.keys(json);//['foo','baz']
```
#### Object.values()
```
	var json = {foo:"1",baz:"2"};
	Object.values(json);//['1','2']
```
### Object.entries()
```
	var json = {foo:"1",baz:"2"};
	Object.entries(json);//[['foo','1'],['baz','2']]
```
