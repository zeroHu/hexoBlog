### 第三章：数组的结构赋值
#### 3.1 为变量赋值
```
es5 声明变量的方式

var a = 1;
var b = 2;
var c = 3;
var sendA = [],
    sendB = [],
    sendC = [];

es6 声明变量的方式

let [a,b,c] = [1,2,3];
let [sendA,sendB,sendC] = [[],[],[]];

//more example
let [head, ...tail] = [1, 2, 3, 4]; head // 1
tail // [2, 3, 4]
```
> 这种写法为模式匹配，只要等号两边相等就可以匹配,解构赋值不仅适用于var命令 , 解构不仅可以用于数组，还可以用于对象。，也适用于let和const命令。

```
let [x,y,z] = new Set(["a","b","c"])
//x : a
```
```
let {a,b} = {foo:"aa",bar:bb}


let { a = {}, b= {} } = {};
```
