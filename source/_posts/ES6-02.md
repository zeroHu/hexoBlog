### 第二章: let和const
#### let基本用法
> 1.let命令所在的代码块内有效，<br>2.let不允许声明重复的变量，<br>3.let不会存在变量提升(如果在声明前使用会报错)

```
{
	let a = 0;
	var b = 1;
}
a //ReferenceErr
b //1
```
#### const基本用法
> 1.const是声明一个只读的常量，也是代码块内
```
if(true){
	const MAX = 5;
}
MAX // Uncaught ReferenceError
```
